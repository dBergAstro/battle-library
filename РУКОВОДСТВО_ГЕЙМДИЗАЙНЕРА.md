## Обзор инструмента

Battle Library Tool — это визуальный инструмент для просмотра и анализа библиотеки боёв из игры Invasion. Позволяет:

- Просматривать все актуальные бои с составами команд
- Анализировать записи (replays) с грейдами и баффами
- Собирать подходящие бои в коллекцию
- Фильтровать по типу, главе и тегам

### Основные страницы

| Страница | URL | Описание |
|----------|-----|----------|
| Библиотека боёв | `/` | Все бои и записи |
| Админ-панель | `/admin` | Загрузка и управление данными |

## Просмотр библиотеки боёв

### Карточка боя содержит:

- **#ID** — уникальный идентификатор боя (gameId)
- **Тип** — Героический (синий) или Титанический (оранжевый)
- **Глава и номер боя** — например "Глава 5, Бой 3"
- **Power Level** — уровень силы (если загружен)
- **Состав команды** — аватары и имена персонажей
- **Тотемы стихий** — для титанических боёв (вода, огонь, земля, свет, тьма)

### Карточка записи (Replay) содержит:

- Те же данные что и бой, плюс:
- **Грейды персонажей** — цветные рамки вокруг аватаров:
  - Фиолетовый — 1-2 фрагмента
  - Оранжевый — 3-6 фрагментов
  - Красный — 7+ фрагментов
- **Основной питомец** — большая иконка справа
- **Покровительство** — маленькие иконки питомцев на персонажах
- **Размер баффа** — значок ↑ с процентом (например ↑100%)
- **Кнопка копирования** — копирует defendersFragments в буфер обмена

## Просмотр записей (Replays)

### Группировка записей

Записи с одинаковым составом команды группируются в одну карточку:
- Показывается бейдж с количеством записей (например "3")
- При нажатии на "+" открывается модалка для выбора конкретной записи

### Информация в модалке выбора:

- Номер боя
- Грейды (цветные круги)
- Тотемы и скилы (для титанических)
- Размер баффа

## Работа с коллекцией

### Добавление в коллекцию

1. Найдите нужный бой или запись
2. Нажмите кнопку **+** на карточке
3. Выберите слот в модальном окне (Глава 1-7, Бой 1-8)
4. Элемент добавится в коллекцию

### Структура коллекции

- 7 глав по 8 боёв = 56 слотов
- Каждый слот может содержать один бой или одну запись
- Записи отмечаются оранжевым цветом и показывают рекомендуемый ID

### Панель коллекции

Открывается кнопкой в правом нижнем углу. Показывает:
- Все заполненные слоты по главам
- Кнопку копирования для записей
- Кнопку удаления элемента
- Кнопку очистки всей коллекции

### Для записей в коллекции отображается:

- Рекомендуемый ID боя (для создания нового боя в базе)
- Размер баффа (↑%)
- Кнопка копирования defendersFragments

## Загрузка и обновление данных

### Доступ к админ-панели

Откройте `/admin` в браузере.

### Таблицы для загрузки

#### 1. Boss List (invasion_boss_list-boss_list)
Основная таблица боёв. Содержит:
- `id` / `gameId` — ID боя
- `label` — глава
- `desc` — номер боя  
- `heroId` — ID главного героя (определяет тип боя)

**Важно:** Бои с gameId ≤ 225 автоматически скрываются как неактуальные.

#### 2. Boss Team (invasion_boss_list-boss_team)
Состав команд противников:
- `bossId` / `bossGameId` — ID боя
- `heroId` — ID персонажа
- `unitId` — альтернативный ID персонажа

#### 3. Boss Level (invasion_boss_list-boss_level)
Уровни сложности:
- `bossId` / `gameId` — ID боя
- `powerLevel` — уровень силы

#### 4. Attack Teams (записи/replays)
Записи атакующих команд с JSON структурой:
```json
{
  "units": [26, 64, 13, 43, 34],
  "petId": 6005,
  "favor": { "13": 6003, "26": 6001 },
  "fragments": { "13": 7, "26": 5 },
  "effects": { "buffName": 100 }
}
```

### Процесс обновления данных

#### Обновление боёв (Boss List + Boss Team + Boss Level):

1. Откройте админ-панель `/admin`
2. В секции "Загрузка таблиц" выберите тип: **Boss List**
3. Загрузите CSV/JSON файл или выберите папку
4. Повторите для **Boss Team** и **Boss Level**
5. Дождитесь завершения загрузки (прогресс-бар)

#### Обновление записей (Replays):

1. В секции "Загрузка таблиц" выберите тип: **Attack Teams**
2. Загрузите файл с записями
3. Укажите основной бафф в секции "Настройки" (ключ из effects)

### Загрузка иконок

1. В секции "Иконки персонажей" выберите категорию:
   - **Герои** (ID 1-999)
   - **Крипы** (ID 1000-3999)
   - **Титаны** (ID 4000-5999)
   - **Питомцы** (ID 6000-6999)
2. Выберите папку с иконками
3. ID извлекается из имени файла (например `titan_big_4003.png` → ID 4003)

### Настройка имён и сортировки

#### Имена персонажей:
- Введите в текстовое поле формат: `ID[tab]имя`
- Эти имена имеют приоритет над встроенными

#### Порядок сортировки:
- Формат: `ID[tab]порядок`
- Поддерживаются дробные значения (4.5 для вставки между 4 и 5)

#### Стихии титанов:
- Формат: `ID стихия очки`
- Стихии: вода, огонь, земля, тьма, свет
- Пороги активации тотемов:
  - Вода/Огонь/Земля: 3+ очков
  - Свет/Тьма: 2+ очков

## Фильтрация и поиск

### Фильтры

| Фильтр | Описание |
|--------|----------|
| Тип боя | Все / Героический / Титанический |
| Глава | Фильтр по номеру главы (1-15+) |
| Источник | Бои / Записи / Всё |

### Поиск по тегам

1. Добавьте теги к боям/записям (кнопка #)
2. Используйте поиск с `#тег` для фильтрации
3. Можно комбинировать несколько тегов

### Поиск по ID

Введите ID боя в поле поиска для быстрого нахождения.

## Советы по работе

1. **Регулярно обновляйте данные** — загружайте новые версии таблиц при изменениях в геймдизайне
2. **Используйте теги** — помечайте интересные бои для быстрого поиска
3. **Проверяйте баффы** — обращайте внимание на размер баффа (↑%) при выборе записей
4. **Копируйте defendersFragments** — используйте для создания новых боёв в редакторе
5. **Следите за грейдами** — цвет рамки показывает уровень прокачки персонажей

## Поддержка

При возникновении проблем:
1. Проверьте формат загружаемых файлов
2. Убедитесь что ID в данных корректны
3. Обновите страницу для получения актуальных данных

## Обзор инструмента

Battle Library Tool позволяет:
- Просматривать все актуальные бои с составами команд
- Анализировать записи с грейдами и баффами
- Помечать бои тегами для быстрого поиска
- Собирать подходящие бои в коллекцию

**Страницы:**
- `/` — Библиотека боёв и записей
- `/admin` — Загрузка данных

---

## Просмотр библиотеки

### Карточка боя показывает:
- **#ID** — идентификатор боя
- **Тип** — Героический (синий) / Титанический (оранжевый)
- **Глава и номер боя**
- **Power Level** — уровень силы
- **Состав команды** — аватары персонажей
- **Тотемы** — для титанических боёв

### Карточка записи (Replay) дополнительно показывает:
- **Грейды** — цветные рамки (фиолетовый 1-2 фрагмента, оранжевый 3-6, красный 7+)
- **Питомец** — большая иконка справа
- **Покровительство** — маленькие иконки питомцев на персонажах
- **Размер баффа** — значок ↑ с процентом
- **Кнопка копирования** — копирует defendersFragments

## Система тегов

Теги — мощный инструмент для организации боёв и записей. Помечайте бои тегами чтобы потом быстро находить нужные.

### Как добавить тег

1. Нажмите кнопку **#** на карточке боя или записи
2. Введите название тега (без #)
3. Нажмите Enter или кнопку добавления
4. Можно добавить несколько тегов одному бою

### Как искать по тегам

В поле поиска введите тег с символом #:
- `#контроль` — найдёт все бои с тегом "контроль"
- `#вода #огонь` — найдёт бои с обоими тегами

### Примеры использования тегов

#### Пример 1: Теги стихий для титанических боёв

Помечайте титанические бои по доминирующим стихиям:

| Тег | Когда ставить |
|-----|---------------|
| `#вода` | Бой с сильным водным тотемом |
| `#огонь` | Бой с огненным тотемом |
| `#земля` | Бой с земляным тотемом |
| `#свет` | Бой со световым тотемом |
| `#тьма` | Бой с тёмным тотемом |
| `#двастихии` | Бой активирует два тотема |

**Сценарий:** Вам нужно найти все титанические бои с водным тотемом.
1. Заранее помечаете такие бои тегом `#вода`
2. Когда нужно — вводите `#вода` в поиск
3. Видите только водные бои

#### Пример 2: Тег #контроль для героических боёв

Помечайте бои с сильными контролирующими составами:

**Когда ставить тег #контроль:**
- В команде есть Джорген, Фобос, Гелиос, Лиан
- Комбинации с массовым оглушением/сном
- Составы, блокирующие ульты противника

**Сценарий:** Ищете примеры сильных контроль-пачек.
1. Просматриваете записи и помечаете интересные тегом `#контроль`
2. Позже вводите `#контроль` в поиск
3. Получаете список всех помеченных составов

#### Ещё идеи для тегов

| Тег | Описание |
|-----|----------|
| `#блендер` | Составы с Себастьяном + К'арх |
| `#танк` | Бои с несколькими танками |
| `#магия` | Магические составы |
| `#физика` | Физические составы |
| `#аое` | Массовый урон |
| `#антихил` | Составы против лечения |
| `#сложный` | Особо сложные бои |
| `#лёгкий` | Простые для прохождения |
| `#баг` | Требует внимания (баг/ошибка) |

### Советы по тегам

1. **Называйте коротко** — `#контроль` лучше чем `#контролирующий_состав`
2. **Используйте единый стиль** — договоритесь в команде о названиях
3. **Комбинируйте теги** — один бой может иметь `#вода`, `#контроль`, `#сложный`
4. **Помечайте сразу** — увидели интересный бой → сразу добавьте тег

## Работа с коллекцией

### Добавление в коллекцию

1. Найдите нужный бой или запись
2. Нажмите **+** на карточке
3. Выберите слот (Глава 1-7, Бой 1-8)

### Панель коллекции

Открывается кнопкой в правом нижнем углу:
- Все слоты по главам (7 глав × 8 боёв)
- Для записей — рекомендуемый ID и размер баффа
- Кнопки копирования и удаления

## Обновление данных

### Важно: как работает загрузка

**Большинство таблиц работают по принципу "добавить или обновить":**
- Если запись с таким ID уже есть → она обновится новыми данными
- Если записи нет → она добавится
- Старые записи, которых нет в новом файле → останутся

**Это значит:** можно загружать только новые или изменённые данные, не нужно каждый раз загружать всё.

**Исключение — Boss Team:** эта таблица требует полной перезагрузки (очистка + загрузка), потому что структура не позволяет определить уникальность записи.

### Что нужно обновлять

#### При добавлении новых боёв:

1. **Boss List** — загрузите файл с новыми боями
   - Можно загрузить только новые записи
   - Существующие обновятся, новые добавятся

2. **Boss Team** — загрузите полный файл составов
   - Сначала нажмите "Очистить Boss Team"
   - Затем загрузите весь файл заново

3. **Boss Level** (если нужен Power Level)
   - Можно загрузить только для новых боёв
   - Или весь файл — существующие обновятся

#### При добавлении новых записей (Replays):

1. **Attack Teams** — загрузите файл с записями
   - Новые добавятся автоматически
   - Существующие (по тому же ID) обновятся

2. **Основной бафф** — укажите ключ баффа в настройках
   - Это ключ из поля `effects` в записях
   - Например: `percentInOutDamageModAndEnergyIncrease_any_99_100_300_99_1000_30`

#### При добавлении новых персонажей:

1. **Иконки** — загрузите папку с новыми иконками
   - Выберите категорию (Герои/Крипы/Титаны/Питомцы)
   - Новые добавятся, существующие обновятся

2. **Имена** (если нужно переименовать)
   - Формат: `ID[tab]имя` на каждой строке
   - Добавляйте только те, что нужно изменить

3. **Порядок сортировки** (если нужен особый порядок)
   - Формат: `ID[tab]порядок`
   - Можно дробные значения (4.5 для вставки между 4 и 5)

#### При изменении стихий титанов:

1. **Стихии титанов** — обновите список
   - Формат: `ID стихия очки`
   - Стихии: вода, огонь, земля, тьма, свет

### Типичные сценарии обновления

#### Сценарий 1: Добавили 10 новых боёв

1. Откройте `/admin`
2. Boss List → загрузите файл только с 10 новыми записями
3. Boss Team → очистите, загрузите полный файл
4. Boss Level → загрузите файл с powerLevel для новых боёв

#### Сценарий 2: Получили новые записи (replays)

1. Откройте `/admin`
2. Attack Teams → загрузите файл с новыми записями
3. Готово — новые записи появятся в библиотеке

#### Сценарий 3: Добавили нового персонажа

1. Откройте `/admin`
2. Выберите категорию иконок (Герои/Титаны/и т.д.)
3. Загрузите папку с иконкой нового персонажа
4. Если нужно имя → добавьте в поле "Имена персонажей"

### Проверка загрузки

После загрузки проверьте статистику в верхней части админ-панели:
- Количество боёв
- Количество записей
- Количество иконок по категориям

## Быстрый справочник

| Действие | Как сделать |
|----------|-------------|
| Найти бой по ID | Введите ID в поиск |
| Найти по тегу | Введите `#тег` в поиск |
| Добавить тег | Кнопка # → ввести тег → Enter |
| Добавить в коллекцию | Кнопка + → выбрать слот |
| Скопировать данные записи | Кнопка копирования на карточке |
| Обновить бои | Админка → Boss List → загрузить файл |
| Обновить записи | Админка → Attack Teams → загрузить файл |

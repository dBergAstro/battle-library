Привет! Нам нужно адаптировать это React приложение для работы внутри Google Apps Script (GAS). Объясняю контекст и задачу.

Контекст
Сейчас приложение работает как полноценный Replit сервис: Express бэкенд → PostgreSQL (Neon) → React фронтенд. Мы хотим перенести его так, чтобы:

Бэкенд = Google Apps Script (Code.js) с Google Sheets как базой данных — уже написан
Фронтенд = этот React проект, но собранный в один HTML файл и деплоимый через GAS
В GAS нет REST API — вместо него используется google.script.run (специальный RPC механизм)
В dev режиме google.script.run надо мокировать чтобы можно было разрабатывать локально
Задача
1. Добавить vite-plugin-singlefile

Установи пакет и настрой Vite чтобы весь JS/CSS инлайнился в один index.html файл:

npm install vite-plugin-singlefile
В vite.config.ts добавь:

// в plugins: [..., viteSingleFile()]

2. Создать файл client/src/lib/gasApi.ts

Этот файл заменяет все вызовы к REST API (/api/battles, /api/replays и т.д.) на вызовы через google.script.run. В prod он вызывает реальный GAS, в dev — мок.

function gsRun<T>(fnName: string, ...args: any[]): Promise<T> {
  return new Promise((resolve, reject) => {
    let runner = (window as any).google.script.run
      .withSuccessHandler((result: any) => {
        if (!result) { reject(new Error(`${fnName} returned null`)); return; }
        if (result.error) { reject(new Error(result.error)); return; }
        resolve(result as T);
      })
      .withFailureHandler((err: any) => reject(new Error(err.message || String(err))));
    runner[fnName](...args);
  });
}
export const gasApi = {
  getBattles:    () => gsRun('getBattles'),
  getReplays:    () => gsRun('getReplays'),      // если нужен отдельный
  getTags:       () => gsRun('getTags'),
  getCollection: () => gsRun('getCollection'),
  getAdminStats: () => gsRun('getAdminStats'),
  saveTag:       (battleGameId: number, tag: string) => gsRun('saveTag', battleGameId, tag),
  deleteTag:     (battleGameId: number, tag: string) => gsRun('deleteTag', battleGameId, tag),
  saveCollectionItem: (item: any)  => gsRun('saveCollectionItem', item),
  deleteCollectionItem: (id: string) => gsRun('deleteCollectionItem', id),
  clearCollection: () => gsRun('clearCollection'),
  adminUpload: (type: string, data: any[]) => gsRun('adminUpload', type, data),
  setMainBuffName: (name: string) => gsRun('setMainBuffName', name),
};

    // ... остальные функции
  };
  
  // установить window.google.script.run с async задержкой

4. Заменить useQuery вызовы

В BattleLibrary.tsx, ReplayLibrary.tsx, AdminPanel.tsx заменить:

// БЫЛО:
const { data, isLoading } = useQuery({ queryKey: ["/api/battles"] });
// СТАЛО:
const [data, setData] = useState(null);
const [isLoading, setIsLoading] = useState(true);
useEffect(() => {
  gasApi.getBattles()
    .then(d => { setData(d); setIsLoading(false); })
    .catch(e => { setError(e.message); setIsLoading(false); });
}, []);
5. Вызвать мок в main.tsx

import { installGasMock } from './lib/gasMock';
installGasMock(); // до ReactDOM.createRoot
